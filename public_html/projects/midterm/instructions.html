Instructions

Learn how to play Head Hunter below

Controls

  • Jump - space
  • Move - WASD
  • Attack - left mouse button
  • Damge Self(Not Recomended) - R

Objectives

Travel through procedurally generated rooms and locate the exit door to advance. Enter the exit door by jumping in front of it by either using 'W' or 'space'.

  1. Move the player left and right using 'A' and 'D' respectively
  2. Jump using 'W' or 'space'
  3. Attack using 'left mouse button'
  4. Defeat an ennemy
  5. Explore the level
  6. Advance through any exit door by jumping in front of it
