I want you to present the pages you like and a description of a minimum of three reasons why you like them in the following format:

The Good

First Example of Web Page With Good Design

Adobe

Adboe utilizes color to effectively draw the user in and compliment their expressive design philosophies. Additionally, adobe presents the user with a robust navigation bar to link to most pages in minimal clicks. Finally, the footer includes links to all Adobe resources to comply with ADA.

Second Example of Web Page With Good Design

Spotify

Spotify utilizes their index page as a landing page prompting the user to download the app. Additionally, the color scheme is simple and provides effective contrast. Finally, the navigation bar is consise as to not overwhel any users.

Third Example of Web Page With Good Design

Soundcloud

Soundcloud effectively delivers new music tailored to the users listening history right as they enter the site. The color scheme is simple and orange pops well against both black and white. Containers help group playlists of content, minimizes the space consumed and maximizing the content provided.

The Bad and the Ugly

For this part of the assignment, I want you to do the same thing you did for the first part, but instead of what you like, you will be describing what you don't like. You must include at least three reasons that the site you don't like has bad design. You can find many examples of bad design on this list of the top 50 worst Web sites .

First Example of Web Page With Bad Design

My Web Page

My website exhibits bad design because its incomplete, lacking any substantial media or information. The website colors could compliment and contrast each other more effectively. The website lacks any elements of visual interest, to help build a cohesive look.

Second Example of Web Page With Bad Design

Yahoo

Yahoo struggles to minimize its purpose to provide clear usefulness. The home page is overwhelming with a search, many navigation options, and news all competing for attention. Many moving containers, ads, and animated images/clips are choppy and distracting.

Third Example of Web Page With Bad Design

Pacific Northwest X-Ray Inc.

This website lacks effective color scheme and contrast as a result. Additionally, every bulleted link has the same image which adds to the confusion as opposed to using different images which would help the user understand what each link directs them towards. Many containers are arbitrarily placed across web pages and lack grid formatting.
